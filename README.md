# AI Job Tacker

Application intended to assist users in managing the job application journey with AI assistance for understaning job requirements.

## Live Production Application

The application is deployed on an optimized Ubuntu 24.04 droplet architecture. Due to active LLM processing and API cost-containment measures, the public registration endpoint is disabled. 

* **Live URL**: [https://aijobtrack.yejoongpaulk.dev](https://aijobtrack.yejoongpaulk.dev)
